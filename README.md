## **～项目概述～**

「袖珍像素体」是基于「k6x8」增补简繁体汉字的6x8（含字间距）点阵字体，适合在空间极度紧张又需要显示汉字的场景使用。

同时也大概是市面上尺寸最小的中文点阵字体。

不过作为代价，可读性和实用性都非常低下，阅读时高度依赖上下文。

若非空间所迫，建议优先考虑更大尺寸的像素字体。

目前本字体项目仍在持续更新，欢迎各位网友提出宝贵意见。
